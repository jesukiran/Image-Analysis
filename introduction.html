Installation (Recommended)¶

Before you start, please check if you have the following requirements:

  • NX-client
  • A script-editor (Atom)
  • An image viewer (Fiji)
  • KLayout
  • Xeneth64
  • SMPlayer

1. Request a client connection to the Imec servers¶

For newcomers, you would need to request ICT to install NX-client on your computers. You can do it by requesting NX-Client in the ICT software portal. Kindly get in touch with helpdesk@imec.be or call the helpline number +32 (0)16 28 13 97!

2. Install a script-editor¶

There are many free-to-use script editors available online. Atom is multi-purpose and serves also as a IDE for Python on your local computer. You can download and follow the instructions to install here: https://atom.io/

3. Install an image viewer¶

For rapid (though not as quick as you would like) viewing of large stacks of image data, you can use Fiji. You can download and follow the instructions to install here: https://imagej.net/Fiji/Downloads

4. Install KLayout¶

For viewing and editing the GDS (Graphic Data System) mask. You can download and follow the instructions to install here: https://www.klayout.de/build.html

5. Install Xeneth64¶

For viewing videos from Xeneth camera (with proprietary format .xvi), you will need their software. Please contact Alejandra: Alejandra.JaureguiUribe@imec.be

6. Install SMPlayer¶

On numerous occasions, you might need to view a large video file and VLC player might not work. For such cases, you can use SMPlayer and the instructions for installation can be found here: https://www.smplayer.info/

Tip

If your scripts are running slow, you can check the server load here (http://ganglia2.imec.be/ganglia/) and run your scripts on a server which has a lower compute load.
